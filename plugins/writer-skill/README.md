# writer-skill

写作风格 Skill 插件。让 AI 写出有个人味道的科技文章，而不是千篇一律的 AI 味。

## 方法论

参考宝玉AI提出的"写作风格 Skill"方法论：不是用提示词去 AI 味，而是给 AI 一份持续更新的"菜谱"，让它学会你的口味。

## 风格特征

- 叙事驱动，像讲故事不像写报告
- 生活化类比解释技术概念
- 口语化表达，自然融入网络用语
- 情感真实，不装全知全能
- 人文落脚，技术最终回到人

## 使用方式

安装插件后，在写作场景中 AI 会自动加载写作风格 Skill，产出符合卡兹克风格的内容。

## 迭代建议

这份 Skill 是初版"菜谱"。按照方法论，应该在实际使用中不断迭代：

1. 让 AI 按 Skill 写一篇文章
2. 手动修改成你满意的样子
3. 对比差异，更新 Skill 规则
4. 重复以上步骤
